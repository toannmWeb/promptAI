# M1-FEATURE — Mode 1 Feature Planning Benchmark

```prompt
mode 1:

Task:
Plan a new feature for an applied project and produce a PRP draft outline.

Feature:
- Name: <feature name>
- Brief: <1-2 sentence feature description>
- User value: <who benefits and why>

Scope:
- Analysis and planning only.
- Do not edit files.
- Read ROADMAP, memory-bank, ADRs and examples relevant to the feature.

Acceptance criteria:
- AC-1: 3 implementation approaches with trade-offs.
- AC-2: Recommendation with reversibility and risk.
- AC-3: PRP outline with testable AC IDs.
- AC-4: Out-of-scope list.
- AC-5: Decision points needing user input.

Output:
Use Mode 1 dense format. Keep it concise enough to be used as a PRP seed.
```
