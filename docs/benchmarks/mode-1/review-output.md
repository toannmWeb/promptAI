# M1-REVIEW — Mode 1 Review Benchmark

```prompt
mode 1:

Task:
Review the following AI output/spec/diff for correctness, completeness and risk.

Content to review:
<paste content or provide file path>

Scope:
- Review only.
- Do not edit files.
- If code/docs are referenced, cite file:line.

Acceptance criteria:
- AC-1: Findings ordered by severity.
- AC-2: Each blocker/major finding has a concrete fix.
- AC-3: Missing tests/verification are identified.
- AC-4: Final verdict: proceed / proceed with fixes / do not proceed.

Output:
Use Mode 1 dense format. Prefer fewer high-signal findings over forced filler.
```
